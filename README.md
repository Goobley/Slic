# Slic: Solar Image Classification using Convolutional Neural Networks

## Requirements
For training:

* NVIDIA GPU
* CUDA 9.0+
* `Python 3`+
* `numpy`
* `matplotlib`
* `scipy`
* `astropy`
* `scikit-image`
* `pandas`
* `tqdm`
* `PyTorch` 0.4+

For prediction:

* `Python 3`+
* `PyTorch` 0.4+
* `numpy`
* `matplotlib`
* `tqdm`

Optional:

* `sunpy`
* `palettable`
